== Description ==

Bank transfert payment made possible with this module as well as any non-virtual payment like (french cheques, cash, ...)
Module de paiement par chèque ou virement bancaire pour le plugin WP e-Commerce.
Permet aussi de recevoir des paiements par cash.
Au moment du réglement le client pourra choisir entre les paiements existant mais aussi le paiement par chèque et virement bancaire. Il recevra les informations (ordre du chèque et adresse postale ou RIB de la banque pour faire son virement).

Dans l'interface, vous verrez les paiements en attente et lorsque que le virement ou le chèque est arrivé, vous pouvez valider la vente et envoyer le produit.

== Installation ==

1. Envoyer `wp-e-commerce-cheques-virement-bancaires` vers le dossier `/wp-content/plugins/`
2. Activer le plugin sur le menu 'Extensions' de WordPress
3. Aller sur 'Réglages -> Boutique -> Paiements' et editer -> 'Paiement Chèque Virement Bancaire'
4. Documentation supplémentaire sur : http://wpcb.fr/cheque-virement
